# Superstore-Sales-Analysis
I analyzed a retail sales dataset with around 10,000 orders
to find what is making profit and what is causing losses.

## Tools I Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI
- Jupyter Notebook

## What I Did
- Cleaned the raw data and fixed date columns
- Created new columns like Shipping Days and Profit Margin
- Found that 18% of orders are loss making
- Built charts to understand sales trends
- Made an interactive dashboard in Power BI

## Key findings
- Technology products make the most profit
- Furniture especially Tables keeps losing money
- Any discount above 20% usually results in a loss
- West region has the best sales and profit
- Sales go up every year in the last 3 months
